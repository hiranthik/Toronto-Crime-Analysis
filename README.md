# Toronto Major Crime Trends Analysis

This project analyzes major crime trends in Toronto over multiple years using Python, pandas, and Seaborn. The goal was to explore how different crime categories have changed over time and to practice data cleaning, aggregation, and visualization techniques.

## What This Project Does

-  Loads and preprocesses a dataset of reported crimes in Toronto

-  Groups crimes by MCI Category and Report Year

-  Aggregates counts for each crime type per year

-  Produces a line plot that visualizes long-term trends across categories

## Tools & Libraries

- Python
- pandas for data manipulation
- matplotlib & Seaborn for visualization
